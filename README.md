# このリポジトリは

ベクトルの合成計算をするためのライブラリ（になる予定）のリポジトリです。
[KonaReflection](https://github.com/pankona/KonaReflection)で、バーのスイングによってボールに加わる力、反射角度の計算に使われる予定です。

# How to build

cmake を使っています。以下のコマンドでビルドできる見込みです。

`cd KonaVector/src`  
`cmake .`  
`make`
